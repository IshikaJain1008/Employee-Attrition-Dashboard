# Employee-Attrition-Dashboard
Attrition Dashboard of company for HR to understand the key factors influencing it
Overview:

This project analyzes employee attrition using key HR metrics and KPIs. The dashboard helps identify trends and factors that influence employee turnover.:

a. Key Insights

b. Overall attrition rate

c. Attrition by department and job role

d .Impact of salary, experience, and tenure

e. Gender and age-based attrition patterns

Tools:

1.Power BI

File:
EmployeesDashboard.pbix

Usage:
Open the file in Power BI Desktop and use the filters to explore attrition across different factors.

Purpose:
To support HR teams in understanding attrition patterns and improving employee retention strategies.
